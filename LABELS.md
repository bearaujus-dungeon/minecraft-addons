# Recommended Labels

Create these labels in GitHub after publishing the repository.

| Label | Purpose |
| --- | --- |
| `needs-triage` | New issue needs maintainer review |
| `bug` | Reproducible bug report |
| `compatibility` | Minecraft, loader, mod, data pack, or server compatibility report |
| `enhancement` | Feature request |
| `question` | Support question |
| `permission` | Permission request |
| `license` | License or reupload question |
| `security` | Security contact request |
| `needs-private-contact` | Security issue needs private follow-up |
| `missing-info` | Reporter needs to provide more details |
| `unsupported-version` | Minecraft version or platform is unsupported |
| `duplicate` | Duplicate issue |
| `wontfix` | Intentional non-goal |

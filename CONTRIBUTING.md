# Contributing

This is a public support repository, not the private source-code repository.

Welcome contributions:

- clear bug reports
- compatibility reports
- reproducible test cases
- feature ideas
- documentation corrections for this public support repo
- permission and licensing questions

Not accepted here:

- addon source-code patches
- copied Minecraft assets
- copied third-party assets
- leaked/private builds
- decompiled code
- reuploaded addon files

## Issue Quality

Good issues include exact versions, logs, reproduction steps, and a small test case. Vague reports may be closed or converted into questions until enough information is available.

## Feedback License

By submitting issues, comments, ideas, suggestions, reproduction steps, translations, text, screenshots, or other contributions to this repository, you give `bearaujus` permission to use, modify, reproduce, and distribute that contribution for Bearaujus addon development, documentation, support, and project pages without payment or obligation.

Only submit content that you have the right to share.

# Bearaujus Minecraft Addons

Public support repository for Minecraft Java Edition mods and data packs created by `bearaujus`.

This repository is for:

- bug reports
- compatibility reports
- feature requests
- questions about supported versions and environments
- reupload, modpack, server, and license permission requests
- links to official Modrinth projects when they are published

This repository is not the source-code repository for the addons. Do not upload addon source code, private builds, leaked files, copied assets, or third-party content here.

## Support Policy

- Minecraft edition: Java Edition
- Minimum supported Minecraft version: `26.1.2`
- Current addon families: Fabric mods and data packs
- Unsupported versions: anything below `26.1.2`
- Unsupported mod loaders/platforms for now: Forge, NeoForge, Quilt, Spigot, Paper, Bukkit

Compatibility is handled per addon. When reporting an issue, always include the exact addon version, exact Minecraft version, loader/platform, and whether the issue happens on client, server, or both.

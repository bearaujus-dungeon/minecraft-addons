# Support

Use GitHub Issues in this repository for public support.

Official issue tracker: https://github.com/bearaujus-dungeon/minecraft-addons
General contact: haryo.assyafah@gmail.com

## Before Opening An Issue

Check:

- you are using Minecraft Java Edition `26.1.2` or newer
- you downloaded the addon from an official source
- you are using the correct addon version for your Minecraft version
- required dependencies are installed
- the issue still happens with the smallest reasonable mod/data pack set

## Include These Details

- addon name and version
- addon type: Fabric mod or data pack
- Minecraft version
- Fabric Loader and Fabric API version, if reporting a Fabric mod
- client/server environment
- operating system
- relevant logs or crash report
- reproduction steps
- expected result
- actual result

## Privacy

Do not post:

- access tokens
- session IDs
- private server addresses
- personal information
- paid/private build files
- copied proprietary assets

If a log contains private information, redact it before uploading.

## Unsupported Requests

These are normally closed:

- support for Minecraft versions below `26.1.2`
- requests for Forge, NeoForge, Quilt, Spigot, Paper, or Bukkit support before those platforms are announced
- bug reports from reuploaded or modified addon builds
- requests to redistribute, mirror, or sell addon files without a permission request

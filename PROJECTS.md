## Released

No public addon has been released yet.

## Planned Project Metadata

Each released addon should include:

- project name
- addon type: Fabric mod or data pack
- official Modrinth URL
- latest supported Minecraft version
- supported environment: client-side, server-side, or client and server
- license
- issue labels or known limitations

## Support Baseline

- Minecraft Java Edition `26.1.2+`
- Fabric mods: Fabric only
- Data packs: Java Edition data packs only
- Older Minecraft versions are not supported
